# 팀명
# 1. 프로젝트 개요 및 서비스 소개
주제 : 여행 일정 추천 및 관리 서비스
사용자 맞춤형 일정 추천을 위한 여행 스타일 조사
장소와 사용자 여행 스타일에 태그를 부여하여 추천 알고리즘 생성
Map API를 활용하여 생성된 일정의 위치 시각화 및 경로 표시
드래그 앤 드롭을 활용한 손쉬운 일정 편집 가능
# 2. 주요기능
